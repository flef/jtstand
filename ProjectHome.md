JTStand is a free scripting environment for data collection,
an open source software, written in Java.<br>It helps you to develop and execute Sequences and to analyze the Results.<br>
<br>
JTStand project has been hosted by <a href='http://jtstand.codehaus.org/'>http://jtstand.codehaus.org/</a> for many years, but it has been moved back to this Google maintained code hosting 7/13/2015.
